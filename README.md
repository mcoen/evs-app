# Run EVS App

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Start the Facility IQ API with `OPENAI_API_KEY` set (AI calls are server-side via Facility IQ API).
3. Run the app:
   `npm run dev`
